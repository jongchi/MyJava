거리두기 개편 하루 앞두고, 확진700명대···권덕철 "엄중한 상황"

사회적 거리두기 개편을 하루 앞둔 30일 방역 당국이 우리나라 신종 코로나바이러스 감염증(코로나19) 신규 환자가 700명대 중반을 넘어섰다며 거리두기 단계 상향 가능성을 열어놨다. 권덕철 보건복지부 장관은 30일 정부세종청사에 열린 중앙재난안전대책본부(중대본) 회의에서 “0시 기준, 국내 발생 확진자는 700명대 중반을 기록할 것으로 보인다. 매우 엄중한 상황이다”라고 말했다.
